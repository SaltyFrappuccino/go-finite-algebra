Таблицы Кэли, конечные поля и сопряжённое пространство V*

Все структуры строятся перебором по таблицам операций на множестве {0, ..., n-1} для порядков 2, 3 и 4.

I. Конечные алгебраические структуры

Одна бинарная операция

Всего таблиц Кэли одной операции - это nn^2; для порядка 4 это уже 416 ~ 4,3 млрд, поэтому ассоциативные таблицы ищутся перебором с отсечениями, а не грубой силой.

Все поля одного порядка лежат в одном классе изоморфизма. Число размеченных таблиц совпадает с n!/|Aut(F)|. Для порядка 4 это 4!/|Aut(F4)| = 24/2 = 12, потому что у F4 есть нетривиальный автоморфизм Фробениуса x -> x^2.

II. Векторное пространство V и сопряжённое V*

Разобранный пример - поле F3, размерность dim V = 3. Берём два базиса, раскладываем вектор и ковектор, строим матрицы перехода и сопряжённый базис в V*. Вся арифметика - по модулю 3.

Базисы и координаты вектора

Базис A (стандартный)

(1, 0, 0) (0, 1, 0) (0, 0, 1)

Базис B

(1, 1, 0) (0, 1, 1) (0, 0, 1)

Вектор v = (1, 2, 0)

[v]A = (1, 2, 0)

[v]B = (1, 1, 2)

Координаты меняются контравариантно - да

Матрицы перехода

T (A -> B)

[1 0 0] [2 1 0] [1 2 1]

T (B -> A)

[1 0 0] [1 1 0] [0 1 1]

Взаимно обратны - да

Сопряжённое пространство V*

Базис V*, сопряжённый к A

(1, 0, 0) (0, 1, 0) (0, 0, 1)

Базис V*, сопряжённый к B

(1, 0, 0) (2, 1, 0) (1, 2, 1)

Условие e_i(b_j) = delta_ij - выполнено

Координаты ковектора

Ковектор w = (1, 1, 1)

[w]A* = (1, 1, 1)

[w]B* = (2, 2, 1)

Координаты меняются ковариантно - да

Изотропные векторы

Форма с единичной матрицей Грама, ненулевые v с <v, v> = 0

(1, 1, 1) (2, 1, 1) (1, 2, 1) (2, 2, 1) (1, 1, 2) (2, 1, 2) (1, 2, 2) (2, 2, 2)

Проверки корректности

Программа проверяет 38 инвариантов, все проходят.

  • порядок 2 - полугруппы >= моноиды >= группы
  • порядок 2 - не моноиды + моноиды = полугруппы
  • порядок 2 - не группы + группы = моноиды
  • порядок 2 - неабелевы + абелевы = группы
  • порядок 2 - не кольца + кольца = полукольца
  • порядок 2 - полукольца с 1, не кольца + кольца с 1 = полукольца с 1
  • порядок 2 - без 1 + с 1 = кольца
  • порядок 2 - не тела + тела = кольца с 1
  • порядок 2 - тела не поля + поля = тела
  • порядок 2 - все найденные поля одного порядка лежат в одном классе
  • порядок 2 - число полей совпадает с n!/|Aut(F)|
  • порядок 3 - полугруппы >= моноиды >= группы
  • порядок 3 - не моноиды + моноиды = полугруппы
  • порядок 3 - не группы + группы = моноиды
  • порядок 3 - неабелевы + абелевы = группы
  • порядок 3 - не кольца + кольца = полукольца
  • порядок 3 - полукольца с 1, не кольца + кольца с 1 = полукольца с 1
  • порядок 3 - без 1 + с 1 = кольца
  • порядок 3 - не тела + тела = кольца с 1
  • порядок 3 - тела не поля + поля = тела
  • порядок 3 - все найденные поля одного порядка лежат в одном классе
  • порядок 3 - число полей совпадает с n!/|Aut(F)|
  • порядок 4 - полугруппы >= моноиды >= группы
  • порядок 4 - не моноиды + моноиды = полугруппы
  • порядок 4 - не группы + группы = моноиды
  • порядок 4 - неабелевы + абелевы = группы
  • порядок 4 - не кольца + кольца = полукольца
  • порядок 4 - полукольца с 1, не кольца + кольца с 1 = полукольца с 1
  • порядок 4 - без 1 + с 1 = кольца
  • порядок 4 - не тела + тела = кольца с 1
  • порядок 4 - тела не поля + поля = тела
  • порядок 4 - все найденные поля одного порядка лежат в одном классе
  • порядок 4 - число полей совпадает с n!/|Aut(F)|
  • оба выбранных базиса имеют полный ранг
  • матрицы A->B и B->A взаимно обратны
  • координаты вектора меняются контравариантно
  • dual(A) и dual(B) удовлетворяют условию сопряженности
